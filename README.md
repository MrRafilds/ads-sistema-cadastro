# 💻 Sistema de Cadastro

Aplicação desktop desenvolvida em Java com o objetivo de gerenciar cadastros de usuários, clientes, fornecedores e produtos, utilizando interface gráfica com Swing.

## 🛠 Tecnologias

* Java
* Java Swing
* NetBeans

## 📌 Funcionalidades

* Cadastro de usuários (nome, login e senha)
* Cadastro de clientes
* Cadastro de fornecedores
* Cadastro de produtos com vínculo a fornecedor
* Edição de registros existentes
* Seleção de dados via ComboBox
* Interface gráfica simples e funcional

## 🚀 Projeto

Projeto desenvolvido em grupo como parte avaliativa da nota P1, com foco na aplicação de conceitos de Programação Orientada a Objetos e desenvolvimento de interfaces gráficas em Java.
